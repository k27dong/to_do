### Demo on Heroku
[ToDo](https://enigmatic-thicket-49092.herokuapp.com/)

### Prerequsites
- Ruby
- Rails
- PostgreSQL
- Bundle
```
$ ruby -v
$ rails -v
$ psql --version
$ pg_config --version
$ bundle -v
```

### Getting Started
```
$ git clone https://github.com/k27dong/to_do.git
$ bundle i
$ rake db:create
$ sudo rake db:migrate
$ rails s -p 3000
```
